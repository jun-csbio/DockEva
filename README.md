# DockEva
DockEva: a fast open-source tool for atom mapping and docking evaluation of symmetric molecules

There are lots of highly symmetric molecules. Correctly evaluateing the ligand docking . 

## Pre-requisite:
    - g++

## Installation:

* Download this repository at https://github.com/jun-csbio/DockEva.git. Then, uncompress it and run the following command lines on Linux System.

~~~
  $ cd DockEva-main
  $ chmod 755 ./install.sh
  $ ./install.sh
~~~

## Run example
~~~
  $ ./dockeva example/crystal.mol2 decoy.mol2
~~~
